#fastdoc 

Office Word 文档生成工具

README.md

功能
===========
1、基于POI结合数据对象参数生成doc/docx文档，基于注解
2、基于Word原生XML文档+模板技术生成doc/docx文档，纯粹的内容填充替换
3、基于POI+模板技术生成doc/docx文档
 a、基于自定义XML的渲染模板生成
 b、解析渲染后的模板使用POI生成Word文档
